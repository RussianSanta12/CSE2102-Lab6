Demo API

Available endpoints:

  • POST /api/password/quality — JSON { "password": "..." }
  • POST /api/email/validate — JSON { "email": "..." }

Examples (curl)

  curl -X POST -H "Content-Type: application/json" -d '{"password":"S3cure!Pass"}' http://localhost:8080/api/password/quality

  curl -X POST -H "Content-Type: application/json" -d '{"email":"user@example.com"}' http://localhost:8080/api/email/validate
